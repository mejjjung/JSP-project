## 💡주요 기능

메인페이지 [**상세보기 - WIKI 이동**](https://github.com/mejjjung/JSP-project.wiki.git)
- MEGA BOX 홈페이지를 참고
- 상영중인 인기 영화 포스터 나열

로그인
- DB값 검증
- 로그인 시 세션(Session) 생성

로그아웃
- 로그아웃 시 세션(Session) 삭제

회원가입
- 회원가입 시 입력한 정보를 DB에 저장

마이페이지
- 회원 정보 변경 기능

회원 탈퇴 
- 회원탈퇴 시 DB에서 정보 삭제
- 탈퇴 시 세션(Session) 삭제

검색 페이지
- 영화 이름 검색 기능
